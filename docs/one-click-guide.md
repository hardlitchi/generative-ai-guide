# ワンクリック・デプロイガイド

このプロジェクトには、難しいコマンドを打たなくても、一瞬でWebサイトを公開するためのスクリプトが用意されています。

---

## 🚀 実行方法

自分のPCのOSに合わせて、以下のファイルを「実行」するだけです。

- **Windowsの方**：
  - フォルダの中にある `deploy.bat` をダブルクリック！
- **Mac / Linuxの方**：
  - ターミナルを開き、`./deploy.sh` と入力して実行！

---

## 🔑 最初に必要なこと

Vercel（公開サーバー）への初回デプロイでは、**「ログイン」**という魔法の儀式が必要です。

1. ターミナルで `npx vercel login` と入力。
2. ブラウザが開くので、自分のアカウントでログイン。
3. これで準備完了です！

---

## 💬 実行中に聞かれること

Vercelから英語で質問されたら、**基本的にはすべて `Enterキー` を押すだけでOKです。**

1. `Set up and deploy? [Y/n]` → **Enter**
2. `Which scope?` → **Enter**
3. `Link to existing project? [y/N]` → **Enter**

---

## 🖼 実行画面のイメージ

実行すると、以下のような黒い画面（ターミナル）が表示され、自動で処理が進みます。

```text
🚀 [1/3] 道具（Node.js）を確認中...
📦 [2/3] 資料をWebサイトに変換中...
🌐 [3/3] Vercelに公開中...
```

### 📢 ここでアドバイス！
黒い画面に長いパス（C:\Users\...）や、難しい英語のメッセージがたくさん出てくるかもしれませんが、**「100%無視して大丈夫！」**です。

自動的に処理が進んでいる証拠なので、最後に URL が表示されるまでリラックスして待ちましょう。☕️

---

## 🎉 公開完了！

最後に表示される **URL (https://...)** をクリックしてください。

あなたのサイトが世界中に公開されています！🎉
友達や同僚に自慢しちゃいましょう！
