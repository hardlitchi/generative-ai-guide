# 生成AIを活用したWebサービス自作ガイド

このプロジェクトは、プログラミング未経験の文系新卒者が、生成AIを使いこなしてWebサービスを無料で公開するための教材セットです。

---

## 📁 フォルダ構成と使い方

### 1. `/slides`（学習用スライド）
- `01_introduction.html`: マインドセット
- `02_tools.html`: 環境構築
- `03_ai_prompts.html`: AIへの頼み方
- `04_deploy.html`: Web公開

---

### 📝 舞台裏：AI開発ログ
このプロジェクトがどのようにAIと一緒に作られたのか、そのプロセスを技術ブログとしてまとめています。
- [AI開発ログ（舞台裏）](blog-ai-dev.html)

---

### 2. `/sample-code`（実践用コード）
日本語コメントとログが過剰なほど親切なサンプルです。
- `hello-ai.js`: ターミナルで動かす最初のプログラム。
- `/simple-web`: ブラウザで動く最小限のWebサイト。

---

### 3. `GEMINI.md`（プロジェクト指針）
このプロジェクトの「憲法」です。
「日本語コメント重視」「小分けのファイル」などのルールが記されています。

---

## 🚀 始め方
1. `slides/01_introduction.html` を読み、ワクワクする。
2. `slides/02_tools.html` を見ながら環境を整える。
3. `sample-code/hello-ai.js` を実行して「動く喜び」を体験する。
4. **`deploy.sh`を実行**して、自分のWebサイトを公開する！
5. 生成AIに「こんなの作りたい！」と相談を始める。
