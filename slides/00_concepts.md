# はじめての「概念理解」

### 〜 専門用語を日常の言葉に置き換えてみよう 〜

---

# 1. 「Webサイト」と「Webアプリ」の違い

初心者がよく迷うポイントですが、実はとってもシンプルです。

### 📄 Webサイト（見るだけのページ）
- **例：** 会社のホームページ、ニュース記事
- **イメージ：** ポスターやパンフレット。内容は決まっていて、見るだけ。

### 📱 Webアプリ（動くページ）
- **例：** 掲示板、日記帳、TODOリスト、SNS
- **イメージ：** 文房具や道具。自分たちが何かを書き込んだり、計算したりできる。

**このガイドでは、「Webアプリ」を作れるようになることを目指します！**

---

# 2. 難しいIT用語を「日常」に例えると？

IT用語は難しく聞こえますが、実は私たちの身の回りにあるものに例えられます。

| IT用語 | 日常の言葉で例えると... |
| :--- | :--- |
| **コード** | 魔法の呪文 / お料理のレシピ |
| **GitHub** | 魔法の本棚 / レシピ本 |
| **Vercel** | 世界中の人が見れる展示会場 |
| **デプロイ** | 本を「本棚」から「展示会場」へ持っていくこと |

---

# 3. つまり、あなたがやることは？

1. AIに**「魔法の呪文（コード）」**を唱えてもらう（書いてもらう）。
2. その呪文を**「魔法の本棚（GitHub）」**に大切にしまう。
3. 本棚から**「展示会場（Vercel）」**に運び出す（デプロイ）。

これだけで、世界中の人があなたの作ったものを見れるようになります！
難しいことは考えず、このイメージだけ持っておけばOKです。
