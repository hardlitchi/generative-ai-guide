# プログラミングは、
# 現代の「お料理」です。

### 〜 英語ができなくても、ITが苦手でも大丈夫 〜

---

# 1. あなたは「お料理」ができますか？

もし、「カレーを作ってください」とAIに頼んだら、AIはレシピを教えてくれます。

### プログラミングも同じです。
- あなたが **「こんなサービスが作りたい」** と日本語で言う。
- 生成AIという **「凄腕のシェフ」** が、魔法の呪文（コード）を書いてくれる。

**あなたは、作りたいものを決める「ディレクター」になればいいのです。**

---

# 2. 英語も、暗記も、いりません

昔のエンジニアは、難しい英語の命令をたくさん覚える必要がありました。

### 今は「通訳」がいます
生成AI（ChatGPTやClaude）は、あなたのたどたどしい日本語を、完璧なコンピューターの言葉に翻訳してくれます。

**必要なのは、プログラミングスキルではなく、あなたの「アイデア」と「日本語」だけです。**

---

# 3. 失敗しても、爆発しません

PCの操作で一番怖いのは「壊れること」ですよね。

- **プログラムは何度書き換えても無料です。**
- **エラーが出ても、画面が真っ暗になるだけです。**
- **AIに「エラーが出た、助けて！」と言えば、数秒で直してくれます。**

お料理と違って、材料が無駄になることもありません。

---

# 4. あなたが手にする「魔法」

このガイドを終える頃には、あなたは以下の魔法を使えるようになります。

1. **自分の名前が入ったURL** を手に入れ、世界に公開する。
2. **AIを部下のように使いこなし**、自分の不便を解決する道具を作る。
3. **「自分にもできる！」** という最強の自信を手に入れる。

**さあ、次の章で「Webアプリの正体」をのぞいてみましょう！**
